# Backend Services

This folder contains the backend services that power the Pip-Boy system.
The backend is responsible for system logic, data access, and hardware
interaction.

The frontend UI communicates with these services via local APIs.

---

## Responsibilities

- Provide system stats (CPU, memory, temperature)
- Manage inventory data
- Serve logs and quest data
- Control audio playback for RADIO
- Interface with Raspberry Pi hardware
